## MUSIC-PLATFORM-BACKEND

# Clone the repository
```
git clone https://github.com/seejn/music-platform-backend.git
cd music-platform-backend
```
# Craete .env file
.env.sample exists for reference

# start db
```
make start_db
```

# project setup
```
make
```

# run project
```
make run
```

# stop project
```
make stop
```

# stop db
```
make stop_db
```
